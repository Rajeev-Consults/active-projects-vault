# Inbox Processing

## Rules
- Decide fast (no overthinking)
- Inbox is not storage
- Every item must be processed once
- Inbox should be empty after processing

---

## Actions

### 🚀 Create Project
If it needs multiple steps  
→ Create project in `03-Active-Projects`  
→ Add tasks to `01-Tasks`

---

### ➕ Add to Existing Project
→ Move to:
- `01-Tasks` (if action)
- `02-Notes` (if info)

---

### 📦 Store as Resource
If useful but not actionable  
→ Move to `04-Resources`

---

### ❌ Delete
If not useful → delete immediately

---

## ⚡ Workflow

1. Read item
2. Decide action
3. Execute immediately
4. Remove from inbox
